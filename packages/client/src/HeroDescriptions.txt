ArcaneClash — Hero Portrait Descriptions
=========================================

Save each generated image into packages/client/src/art/ using the EXACT
filename shown (png, jpg, or webp all work). Restart "npm run dev" after
adding files. Heroes without an image keep the plain colored circle.

Format: SQUARE image (e.g. 1024x1024) with the face/bust centered — the
portrait is clipped to a circle on the board, so edges get cut off.

Append this style suffix to every prompt for a coherent set:
anime style, vibrant cel-shaded digital illustration, dramatic fantasy
lighting, painterly background, character bust portrait, no text,
no watermark
