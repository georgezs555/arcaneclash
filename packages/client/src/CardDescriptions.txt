ArcaneClash — Card Art Descriptions
====================================

Save each generated image into packages/client/src/art/ using the EXACT
filename shown (png, jpg, or webp all work). Restart "npm run dev" after
adding files. Cards without an image keep the placeholder art.

Format: portrait orientation, roughly 3:4 (e.g. 768x1024), subject centered,
no text in the image.

Append this style suffix to every prompt for a coherent set:
anime style, vibrant cel-shaded digital illustration, dramatic fantasy
lighting, painterly background, trading card game art, portrait composition,
no text, no watermark
